Border Radius

Border Radius adalah properti css 3 membuat kotak yg ujung" menjadi tumpul. Semakin besar border-radius nya maka ujung dari kotaknya akan tumpul.

Contoh Penggunaan border-radius :

  1. Border-radius biasa

    Hello World
    Script CSS : 
    
        div {
            width: 300px;
            height: 300px;
            background-color: salmon;
            line-height: 300px;
            text-align: center;
            color: white;
        }

        .contoh {
            border-radius: 4px;
        }

  2. Border-radius lingkaran

    Hello World
    Script CSS : 
    
        div {
            width: 300px;
            height: 300px;
            background-color: salmon;
            line-height: 300px;
            text-align: center;
            color: white;
        }

        .lingkaran {
            border-radius: 50%;
        }

  3. Border-radius satu sisi

    Property border-radius bisa digunakan hanya untuk beberapa sisi saja seperti diatas kanan, siatas kiri, dibawah kanan, dan dibawah kiri.

    Berikut macam-macam property-nya :

    • border-top-right-radius : membuat border radius pada sisi kanan atas
    • border-top-left-radius : membuat border radius pada sisi kiri atas
    • border-bottom-right-radius : membuat border radius pada sisi kanan bawah
    • border-bottom-left-radius : membuat border radius pada sisi kiri bawah

    Hello World
    Script CSS : 
    
        div {
            width: 300px;
            height: 300px;
            background-color: salmon;
            line-height: 300px;
            text-align: center;
            color: white;
        }

        .satu-sisi {
            border-top-right-radius: 50%;
        }

  6. Border-radius Shorthand

    Shorthand border-radius memiliki urutan seperti margin padding dan shorthand lainnya, yaitu seperti arah jarum jam. Berikut Script Shorthand dari border-radius :

    
    border-radius: kiri-atas kanan-atas kanan-atas kiri-bawah;
    border-radius: 0 50% 50% 50%;
