RGBa & HSLa

RGBa & HSLa adalah nilai property warna untuk menambahkan warna sama seperti property rgb namun ada penambahan "a" di akhir yg berfunsi sbg opacity dari warna. Funsi "a" itu jg sama seperti property opacity yg memberikan transparansi namun hanya pada warna.

Berikut Struktur property dengan nilai rgba & hsla :

background-color: rgba(red, green, blue, alpha); background-color: hsla(hue, saturation, lightness, alpha);

    Keterangan :

  • red = nilai warna merah
  • green = nilai warna hijau
  • blue = nilai warna biru
  • hue = gradiasi warna
  • saturation = intensitas/kepekatan warna
  • light = kecerahan warna
  • alpha = nilai opacity dari suatu warna

Contoh Penggunaan :

  • Solusi opacity & penggunaan rgba :

    Script CSS :

    .satu {
    height: 300px;
    width: 300px;
    background-color: rgba(250, 128, 114, 0.5);
    position: relative;

}
    
.dua {
    width: 100px;
    height: 100px;
    background-color: blue;
    position: absolute;
    right: -50px;
    bottom: -50px;
}

  • Solusi opacity & penggunaan hsla :

    Script CSS :

    .satu {
    height: 300px;
    width: 300px;    
    background-color: hsla(24, 100%, 68%, 0.5);
    position: relative;

}
    
.dua {
    width: 100px;
    height: 100px;
    background-color: blue;
    position: absolute;
    right: -50px;
    bottom: -50px;
}
