Vendor Prefix

Dari kata prefix yang berarti awalan maksudnya kita akan menambahkan awalan pada property CSS 3 sedangkan vendor adalah browser yang bersangkutan.

Vendor prefix adalah sintaks khusus pada CSS 3 yang harus ditambahkan pada property CSS 3. Jadi ada beberapa property CSS 3 yang tidak jalan di beberapa browser dan Vendo Prefix ini berguna untuk itu. Sintaks khusus tersebut spesifik atau berbeda-beda untuk setiap browser/vendor.

Berikut cara menambahkannya :

  • -webkit- [property CSS 3] : Chrome, Safari, Opera versi terbaru.
  • -moz- [property CSS 3] : Firefox.
  • -ms- [property CSS 3] : Internet Explorer.
  • -o- [property CSS 3] : Opera Versi Lama.

Contoh Penggunaan :

Pada chrome gambar di bawah ini dapat berubah karena chrome sudah bisa pakai property filter ini. Akan tetapi, pada browser Internet Explorer gambar ini tidak berubah menjadi hitam putih.

Kondisi Awal

Script CSS Kondisi Awal :

img {
    width: 200px;
    margin: auto;
    display: block;
}

Kondisi Akhir

Script CSS Kondisi Akhir :

img {
    width: 200px;
    margin: auto;
    display: block;
    filter: grayscale(100%);
}
