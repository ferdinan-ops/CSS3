Box Shadow

Box Shadow adalah property css 3 yang berfungsi untuk memberikan bayangan dari elemen HTML yang kita buat

Struktur dan 4 nilai yg harus ada di box shadow :

box-shadow: (x-offset) (y-offset) (blur) (color);

atau

box-shadow: (inset) (x-offset) (y-offset) (blur) (spread) (color);

Keterangan :

  • x-offset = utk mengatur jarak bayangan pada sumbu horizontal yang nilainya numerik.
  • y-offset = utk mengatur jarak bayangan pada sumbu vertikal yang nilainya numerik.
  • blur = utk mengatur seberapa besar blur pada bayangan
  • color = utk memberi warna bayangan
  • inset = membuat bayangan berada didalm kotak
  • spread = untuk ukuran atau size dari bayangannya.

*note: teks bewarna kuning merupakan hal yg membuat model itu terjadi.

Penggunaan box-shadow property :

  • box-shadow biasa dengan warna solid

    Script CSS :

    .contoh {
    width: 300px;
    height: 300px;
    background-color: lightgreen;
    margin: 50px auto;
    box-shadow: 5px 5px 5px black;
}

  • box-shadow dengan bayangan ke arah kiri dan warna solid

    Script CSS :

    .contoh {
    width: 300px;
    height: 300px;
    background-color: lightgreen;
    margin: 50px auto;
    box-shadow: -5px -5px 5px black;
}

  • box-shadow yang biasa dipakai

    Script CSS :

    .contoh {
    width: 300px;
    height: 300px;
    background-color: lightgreen;
    margin: 50px auto;
    box-shadow: 5px 5px 5px rgba(0, 0, 0, 0.65);
}

  • box-shadow pakai nilai spread

    Script CSS :

    .contoh {
    width: 300px;
    height: 300px;
    background-color: lightgreen;
    margin: 50px auto;
    box-shadow: 5px 5px 10px 5px rgba(0, 0, 0, 0.65);
}

  • box-shadow pakai nilai spread dengan arah bayangan dari bawah

    Script CSS :

    .contoh {
    width: 300px;
    height: 300px;
    background-color: lightgreen;
    margin: 50px auto;
    box-shadow: 0px 30px 10px -10px rgba(0, 0, 0, 0.65);
    transition: .3s;
}
.contoh:hover {
    margin-top: 40px;
    box-shadow: 0px 80px 15px -40px rgba(0, 0, 0, 0.65);
}

  • box-shadow pakai nilai inset

    Script CSS :

    .contoh {
    width: 300px;
    height: 300px;
    background-color: lightgreen;
    margin: 50px auto;
    box-shadow: inset 0px 0px 5px 5px rgba(0, 0, 0, 0.65);
}

  • Multiple box-shadow

    Script CSS :

    .contoh {
    width: 300px;
    height: 300px;
    background-color: lightgreen;
    margin: 50px auto;
    box-shadow: 5px 6px 5px rgba(255, 0, 0, 0.5), -5px -5px 5px rgba(0, 0, 255, 0.5);
}

  • Multiple box-shadow : multiple border

    Script CSS :

    .contoh {
    width: 300px;
    height: 300px;
    background-color: lightgreen;
    margin: 50px auto;
    box-shadow: 0px 0px 0px 5px lightcoral, 0px 0px 0px 10px lightblue, 0px 0px 0px 15px rebeccapurple;
    border-radius: 50%;
}
