Gradient

Gradient adalah teknik untuk memberi warna gradiasi pada elemen yang dimiliki. Gradient memiliki 2 jenis yaitu linear-gradient dan radial-gradient.

Macam-macam Gradient :

  • linear-gradient

    linear-gradient memberikan gradiasi warna secara garis lurus. linear-gradient bukan property css3, melainkan value untuk background-image dan background.

    Sintaks Sederhana linear-gradient :

    background: linear-gradient(warna awal, warna akhir);

    Sintaks linear-gradient Multiple Color :

    background: linear-gradient(warna-1, warna-2, ... , warna-n);

    Sintaks linear-gradient lengkap dengan arah dan sudut :

    background: linear-gradient(sudut | arah , warna-1, warna-2, ... , warna-n);

    Value dari sudut adalah derajat atau degree, contohnya 45deg, 90deg dll.

    Value dari arah :

    • to bottom : dari atas ke bawah.
    • to top : dari bawah ke atas.
    • to left : dari kanan ke kiri.
    • to right : dari kiri ke kanan
    • to left top : dari bawah kanan ke atas kiri.
    • to right top : dari bawah kiri ke atas kanan.
    • to left bottom : dari kanan atas ke kiri bawah.
    • to right bottom : dari kiri atas ke kanan bawah.

  • radial-gradient

    radial-gradient memberikan gradiasi warna secara lingkaran. radial-gradient bukan property css3, melainkan value untuk background-image dan background.

    radial-gradient memiliki banyak nilai, yaitu :

    • Value dari tipe:

      • ellipse : tipe default dari radial-gradient.
      • circle

    • Value dari jangkauan. Maksudnya adalah jangkauan dari warna gradiasi dapat diatur. Jangkauan juga merupakan jarak dari pusat gradient ke sisi terluarnya.

      Vakue dari jangkauan :

      1. closest-corner : jarak dari pusat gradient ke sudut terdekat elemen.
      2. closest-side : jarak dari pusat gradient ke sisi terdekat elemen.
      3. farthest-corner : jarak dari pusat gradient ke sudut terjauh elemen.
      4. farthest-side : jarak dari pusat gradient ke sisi terjauh.

    • Mengatur Posisi :

      • posisi-y. Value dalam satuan px:

        • top
        • center
        • bottom

      • posisi-x. Value dalam satuan px:

        • left
        • center
        • right

    Macam-macam Sintaks radial-gradient :

    • Sintaks Sederhana radial-gradient :

      background: radial-gradient(warna awal, warna akhir);

    • Sintaks radial-gradient Multiple Color :

      background: radial-gradient(warna-1, warna-2, ... , warna-n);

    • Sintaks radial-gradient lengkap dengan arah dan sudut :

      background: radial-gradient([tipe] [jangkauan] 
                                    at [posisi-y] [posisi-x], 
                                    warna-1, 
                                    warna-2,
                                    ... , 
                                    warna-n);

Contoh Penggunaan linear-gradient:

  1. Gradient Sederhana

    div {
    width: 600px;
    height: 300px;
    margin: auto;
    background-image: linear-gradient(red, blue);
}

  2. Gradient Multiple Color

    div {
    width: 600px;
    height: 300px;
    margin: auto;
    background-image: linear-gradient(red, #f0d606, rgba(255, 255, 0, 1), green);
}

  3. Gradient dengan nilai sudut

    div {
    width: 600px;
    height: 300px;
    margin: auto;
    background-image: linear-gradient(90deg, red, blue);
}
    div {
    width: 600px;
    height: 300px;
    margin: auto;
    background-image: linear-gradient(-45deg, red, blue);
}

  4. Gradient dengan nilai arah

    div {
    width: 600px;
    height: 300px;
    margin: auto;
    background-image: linear-gradient(to top, red, blue);
}
    div {
    width: 600px;
    height: 300px;
    margin: auto;
    background-image: linear-gradient(to left bottom, red, blue);
}
    div {
    width: 600px;
    height: 300px;
    margin: auto;
    background-image: linear-gradient(to bottom, red, blue);
}
    div {
    width: 600px;
    height: 300px;
    margin: auto;
    background-image: linear-gradient(to left, red, blue);
}
    div {
    width: 600px;
    height: 300px;
    margin: auto;
    background-image: linear-gradient(to right, red, blue);
}
    div {
    width: 600px;
    height: 300px;
    margin: auto;
    background-image: linear-gradient(to left top, red, blue);
}
    div {
    width: 600px;
    height: 300px;
    margin: auto;
    background-image: linear-gradient(to right top, red, blue);
}
    div {
    width: 600px;
    height: 300px;
    margin: auto;
    background-image: linear-gradient(to right bottom, red, blue);
}

  5. Gradient Color Stop

    div {
    width: 600px;
    height: 300px;
    margin: auto;
    background-image: linear-gradient(to right, red 63%, blue 80%);
}

Contoh Penggunaan radial-gradient:

  1. Gradient Sederhana

    div {
    width: 600px;
    height: 300px;
    margin: auto;
    background-image: radial-gradient(red, blue);
}

  2. Gradient Multiple Color

    div {
    width: 600px;
    height: 300px;
    margin: auto;
    background-image: radial-gradient(red, blue, green, yellow);
}

  3. Gradient Stop Color

    div {
    width: 600px;
    height: 300px;
    margin: auto;
    background-image: radial-gradient(red 10%, blue 20%, green 80%);
}

  4. Gradient dengan tipe :

    • Gradient tipe "ellipse"

      div {
    width: 600px;
    height: 300px;
    margin: auto;
    background: radial-gradient(ellipse, red, blue);
}

    • Gradient tipe "circle"

      div {
    width: 600px;
    height: 300px;
    margin: auto;
    background: radial-gradient(circle, red, blue);
}

  5. Gradient dengan jangkauan :

    • Gradient jangkauan "closest-corner"

      div {
    width: 600px;
    height: 300px;
    margin: auto;
    background: radial-gradient(circle closest-corner, red, blue);
}

    • Gradient jangkauan "closest-side"

      div {
    width: 600px;
    height: 300px;
    margin: auto;
    background: radial-gradient(circle closest-side, red, blue);
}

    • Gradient jangkauan "farthest-corner"

      div {
    width: 600px;
    height: 300px;
    margin: auto;
    background: radial-gradient(circle farthest-corner, red, blue);
}

    • Gradient jangkauan "farthest-side"

      div {
    width: 600px;
    height: 300px;
    margin: auto;
    background: radial-gradient(circle farthest-side, red, blue);
}

    • Gradient dengan posisi

      div {
    width: 600px;
    height: 300px;
    margin: auto;
    background: radial-gradient(circle farthest-side at top left, red, blue);
}
