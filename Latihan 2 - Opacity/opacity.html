Opacity

Property opacity berguna utk membuat elemen menjadi transparan tergantung berapa nilai yang kita ingin buat. *note: nilai 0 pada opacity akan membuat elemen menjadi tak terlihat dan nilai 1 membuat elemen tidak memiliki opacity.

Contoh Penggunaan opacity :

  1. Opacity 1 elemen

    
    .satu {
        width: 300px;
        height: 300px;
        background-color: salmon;
        /* ini pembuat model diatas /
        opacity: 0.5;
    }

  2. Opacity 2 elemen




    
    .satu {
        width: 300px;
        height: 300px;
        background-color: salmon;
        /* ini pembuat model diatas /
        opacity: 0.5;
        position: relative;
    }

    .dua {
        width: 100px;
        height: 100px;
        background-color: blue;
        position: absolute;
        right: -50px;
        bottom: -50px;
    }

  3. Opacity 2 elemen, kotak biru transparansi thdp kotak oren yg transparan.




    
    .satu {
        width: 300px;
        height: 300px;
        background-color: salmon;
        opacity: 0.5;
        position: relative;
    }
        
    .dua {
        width: 100px;
        height: 100px;
        background-color: blue;
        position: absolute;
        right: -50px;
        bottom: -50px;
        /* ini pembuat model diatas /
        opacity: 0.5; 
    }

  4. Opacity 2 elemen, kotak biru transparansi thdp kotak oren yg tak transparan




    
    .satu {
        width: 300px;
        height: 300px;
        background-color: salmon;
        /* ini pembuat model diatas /
        opacity: 1;
        position: relative;
    }
        
    .dua {
        width: 100px;
        height: 100px;
        background-color: blue;
        position: absolute;
        right: -50px;
        bottom: -50px;
        /* ini pembuat model diatas /
        opacity: 0.5; 
    }

  5. Opacity 2 elemen, kotak biru tdk transparan dan kotak oren yg transparan

    • Masalah



      
    .satu {
        width: 300px;
        height: 300px;
        background-color: salmon;
        /* ini pembuat model diatas /
        opacity: 1;
        position: relative;
    }
        
    .dua {
        width: 100px;
        height: 100px;
        background-color: blue;
        position: absolute;
        right: -50px;
        bottom: -50px;
        /* ini pembuat model diatas /
        opacity: 0.5; 
    }

      Bila anda melihat dua kotak diatas terdapat perbedaan antara contoh yg ke 4 dan yg ini. Disini kita ingin kotak biru tidak transaparan dgn memberi nilai opacity: 1; untuk membuatnya tdk transparan dan kotak orens diberi nilai opacity: 0.5; utk membuatnya transparan. Namun hal itu tidak terjadi, malah membuat dua"-nya menjadi transparan itulah masalah dari opacity.
