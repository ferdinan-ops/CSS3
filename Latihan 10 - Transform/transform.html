Transform

Transfo adalah property CSS 3 yang memungkinkan kita dapat memanipulasi format visual dari elemen HTML yang kita miliki. Formatnya sendiri beragam yaitu ukuran, bentuk dll.

Syntax penulisan property transform: transform: (fungsi);

Propety transformasi memiliki 2 jenis, yaitu :

  • 2D (Dua Dimensi)
  • 3D (Tiga Dimensi)

Fungsi-fungsi yang ada pada transformasi :

  • scale : memperbesar / memperkecil elemen. Cara penulisannya : scale(angka), scaleX(angka), scaleY(angka).
  • rotate : merotasikan / memutar elemen. Cara penulisannya : rotate(deg), rotateX(deg), rotateY(deg).
  • skew : membuat condong / miring. Cara penulisannya : skew(deg), skewX(deg), skewY(deg).
  • translate : mengubah posisi. Cara penulisannya : translate(px), translateX(px), translateY(px)

Contoh Penggunaan

  • Penggunaan Values Scale

    • Scale(angka) Values

      Script CSS :

      .contoh{
    width: 200px;
    height: 200px;
    background-color: lightblue;
    margin: 50px auto;
    transition: 1s;
}

.contoh:hover{transform: scale(1.2);}

    • ScaleX(angka) Values

      Script CSS :

      .contoh:hover{transform: scaleX(1.5);}

    • ScaleY(angka) Values

      Script CSS :

      .contoh:hover{transform: scaleY(1.2);}

    • Scale Gabungan Values

      Script CSS :

      .contoh:hover{
  transform: scale(1.5, 1.2);
  /* or
    transform: scaleX(1.5) scaleY(1.2);
     */
}

    transition: 1s;
}

    • RotateY(deg) Values
