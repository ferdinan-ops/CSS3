Font Face

Font face merupakan property CSS 3 yang digunakan untuk mengelola font pada CSS 3, sama seperti property font-family namun font-family memiliki keterbatasan yaitu font yang kita pakai harus ada pada komputer kita dan komputer yang membuka website kita.

Sintaks @font-face :

@font-family{
    font-family: 'namaFontbaru';
    src: url('lokasi/file/fonBaru.eot');
    src: url('lokasi/file/fontBaru.eot?#iefix') format(embedded-opentype),
        url('lokasi/file/fontBaru.woff2') format('woff2'),
        url('lokasi/file/fontBaru.woff') format('woff'),
        url('lokasi/file/fontBaru.ttf') format('truetype'),
        url('lokasi/file/fontBaru.svg#svgFontName') format('svg');
}

Keterangan :

namaFontbaru : font yang telah diunduh dan disimpan dalam folder.
src : property untuk mencari file font dalam folder.
url('') : lokasi tempat font berada.
eot,woff2,ttf,svg : ekstensi file dari font.

Dari semua jenis ekstensi font yang ada yang paling sering ada ialah ttf/totf dan woff. Namun, jenis font ini membuat website menjadi lambat dibanding dengan font woff. Oleh sebab itu maka ekstensi file font woff lebih baik digunakan daripada ekstensi file ttf.

Contoh Penggunaan Font Face :

  • Menggunakan Google Font

    Cara ini digunakan apabila kita tidak ingin khawatir karena font harus ada pada user/client. Untuk menggunakan cara ini kunjungi websitenya yaitu https://fonts.google.com/. Langkah selanjutnya pilih font dan salin link atau gunakan @import lalu salin CSS Rules untuk property css nya.

    Berikut hasilnya menggunakan font Comforter Brush.

    Hello World

    Script :
    h1 {
    font-family: 'Comforter Brush', cursive;
    text-align: center;
}
    • <link rel="preconnect" href="https://fonts.googleapis.com"> <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin> <link href="https://fonts.googleapis.com/css2?family=Comforter+Brush&display=swap" rel="stylesheet">

  • Menggunakan @font-face

    Hello World

    Script :
    @font-face {
    font-family: "Alex Brush";
    src: url(AlexBrush-Regular.ttf) format('trutype');
}

h1 {
    font-family: 'Alex Brush', arial;
    text-align: center;
}
