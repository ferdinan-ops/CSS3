Filter

Filter merupakan property CSS 3 yang digunakan untuk memberikan efek visual pada gambar ,background atau border.

Syntax property filter : filter: [tipe-filter]

Tipe Filter :

  • blur : px.
  • brightness : angka atau %.
  • contrast : angka atau %.
  • saturate : angka atau %.
  • grayscale : angka atau %.
  • sepia : angka atau %.
  • hue-rotat : deg dalam artian derajat searah jarum jam.
  • invert : angka atau %.
  • opacity : angka atau %.
  • drop-shadow : sama seperti box-shadow.
  • gabungan dua atau lebih tipe.

Berikut dibawah ini merupakan gambar pemandangan yang luar biasa tanpa ada filter apapun pada gambar ini. Gambar ini adalah gambar normal yang sebelum di filter dengan tipe filter apapun. Nah, gambar ini yang akan diberikan property filter dari semua tipe filter yang ada. 

img {margin: auto; width: 500px; display: block;}

Contoh Penggunaan :

  • Penggunaan blur value

    Gambar dibawah ini merupakan gambar yang telah ditambahkan property filter tipe blur sebanyak 5px. Diberikan 5px karena jika sangat besar maka gambarnya semakin tidak kelihatan.

    img {margin: 20px auto; filter: blur(5px); width: 500px; display: block;}

  • Penggunaan brightness value

    Gambar dibawah ini merupakan gambar yang telah ditambahkan property filter tipe brightness sebanyak 2. Diberikan 2 karena jika sangat besar maka gambarnya akan semakin terang, memberikan nilai angka pada tipe brightness berfungsi untuk mengatur gambar untuk lebih cerah.

    img {margin: 20px auto; filter: brightness(2); width: 500px; display: block;}

    Gambar dibawah ini merupakan gambar yang telah ditambahkan property filter tipe brightness sebanyak 50%. Diberikan 50% karena jika semakin besar maka akan semakin terang dan jika semakin kecil maka gambar akan menjadi hitam gelap, memberikan nilai % pada tipe brightness berfungsi untuk mengatur gambar untuk lebih gelap.. Untuk default dari tipe brightness ini yaitu 1 pada angka dan 100% pada %.

    img {margin: 20px auto; filter: brightness(50%); width: 500px; display: block;}

  • Penggunaan contrast value

    img {margin: 20px auto; filter: contrast(2); width: 500px; display: block;}
    img {margin: 20px auto; filter: contrast(50%); width: 500px; display: block;}

  • Penggunaan saturate value

    img {margin: 20px auto; filter: saturate(3); width: 500px; display: block;}
    img {margin: 20px auto; filter: saturate(10%); width: 500px; display: block;}

  • Penggunaan grayscale value

    img {margin: 20px auto; filter: grayscale(2); width: 500px; display: block;}

    Penggunaan grayscale value % memiliki keunikan dengan tipe lainnya semakin kecil nilainya maka semakin ke default tingkatannya dan semangkin besar maka semakin gelap sedangkan value angka tida bisa ke default kecuali nilai 0. Saran lebih baik gunakan %.

    img {margin: 20px auto; filter: grayscale(50%); width: 500px; display: block;}

  • Penggunaan sepia value

    img {margin: 20px auto; filter: sepia(2); width: 500px; display: block;}
    img {margin: 20px auto; filter: sepia(50%); width: 500px; display: block;}

  • Penggunaan hue-rotate value

    img {margin: 20px auto; filter: hue-rotate(100deg); width: 500px; display: block;}

  • Penggunaan invert value

    img {margin: 20px auto; filter: invert(2); width: 500px; display: block;}
    img {margin: 20px auto; filter: invert(70%); width: 500px; display: block;}

  • Penggunaan opacity value

    img {margin: 20px auto; filter: opacity(.7); width: 500px; display: block;}
    img {margin: 20px auto; filter: opacity(50%); width: 500px; display: block;}

  • Penggunaan drop-shadow value

    img {margin: 20px auto; filter: drop-shadow(.7); width: 500px; display: block;}

  • Penggunaan 2 atau lebih value

    img {margin: 20px auto; filter: blur(5px) grayscale(100%); width: 500px; display: block;}
